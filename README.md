# Requirements
- Python 3.4+
- numpy
- PyQt5
- C++ Compiler

# Installation
## Arch Linux
```bash
yaourt -S urh
```

## From pip
```bash
pip3 install urh
```

## From cloned repository
```bash
git clone https://github.com/jopohl/urh/
cd urh
sudo python setup.py install
```

# Running from source
To execute urh without installation, just run:
```bash
git clone https://github.com/jopohl/urh/
cd urh/bin
./urh
```

Note, on first use the C++ extensions will be built.
